# VTYS_CraftStore_proje

Bu proje, **BMT311 – Veritabanı ve Yönetim Sistemleri** dersi kapsamında geliştirilmiş,
el yapımı ürün satan satıcılar için tasarlanmış bir **Craft Store (mini e-ticaret)** sistemidir.

Proje; veritabanı tasarımı, SQL scriptleri, trigger ve stored procedure kullanımı ile
Java tabanlı bir masaüstü uygulamasını birlikte içermektedir.

---

##  Projenin Amacı

- El yapımı ürün satan **satıcıların (vendor)** ürünlerini yönetebilmesi  
- **Müşterilerin** sipariş oluşturabilmesi  
- Sipariş sonrası **stokların otomatik güncellenmesi**  
- Satışlara yönelik **raporların** alınabilmesi  
- Veritabanı üzerinde **trigger** ve **procedure** kullanımının gösterilmesi  

---

##  Kullanılan Teknolojiler

- **Java (Swing)** – Masaüstü arayüz
- **PostgreSQL** – Veritabanı
- **pgAdmin 4** – Veritabanı yönetimi
- **SQL / PLpgSQL** – Trigger ve stored procedure
- **Maven** – Proje ve bağımlılık yönetimi
- **Git & GitHub** – Versiyon kontrolü
